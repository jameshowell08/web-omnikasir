# Design Language
Hereâ€™s a general guide on how to use the colors for components:

- **Backgrounds:** Use **pure white** or **off-white** for main content areas and **light gray** or **off-white** for subtle section separation. If you're considering a dark mode, flip this, using **off-black** or **dark gray** for backgrounds and **light gray** or **pure white** for text.

- **Text:** **Off-black** or **dark gray** for primary text. **Medium gray** for secondary text (e.g., descriptions, captions). **Light gray** for very subtle or disabled text.

- **Borders & Dividers:** **Light gray** or **medium gray.**

- **Interactive Elements (Buttons, Links):** Your chosen hint of color for primary actions. **Dark gray** or **off-black** for secondary or ghost buttons. Hover states can use a lighter shade of the accent color or a subtle **light gray** background.

- **Icons:** Primarily **dark gray** or **off-black**, with the hint of color for interactive or highlighted icons.
